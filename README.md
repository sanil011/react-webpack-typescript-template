# React + TypeScript + Webpack (Learning Template)

A **minimal React + TypeScript project built with Webpack from scratch**.

This project is designed purely for **learning purposes**. The goal is to understand what tools like **Webpack, Babel, and TypeScript** actually do behind the scenes instead of relying on frameworks that hide the configuration.

Modern tools like **Vite**, **Next.js**, and **Create React App** abstract most of the setup. This template exposes everything so you can clearly see how the **React build pipeline works**.

---

# What This Project Demonstrates

This template helps you understand the core parts of a modern React setup:

- How **Webpack bundles a React application**
- How **Babel transpiles modern JavaScript and JSX**
- How **TypeScript integrates with React**
- How **loaders process different file types**
- How **Hot Reloading works with React Fast Refresh**
- How **ESLint and Prettier maintain code quality**

---

# Tech Stack

| Tool | Purpose |
|-----|------|
| React | UI Library |
| TypeScript | Static typing |
| Webpack 5 | Module bundler |
| Babel | JavaScript transpiler |
| ESLint | Code linting |
| Prettier | Code formatting |
| React Fast Refresh | Hot reloading |

# Getting Started

### Install Dependencies

```bash
yarn install
```  
### Start Development Server
```bash
yarn start
```  
### Build for Production
```bash
yarn build
```
### Code Quality
```bash
yarn lint
```
### Format Code
```bash
yarn format
```  

# Why this exists
Modern tools like Vite and Next.js make development easier, but they also hide how things work internally. This project is meant to explore the fundamentals and understand what those tools automate for us.
Use it as a sandbox to experiment with Webpack configuration and understand the React build pipeline.
